# Laptop Shop Demo

Đây là dự án trang web bán laptop được xây dựng bằng **Spring Boot** theo mô hình **MVC** với **Server-Side Rendering**.

## Công nghệ sử dụng

- **Backend:** Spring Boot
- **Mô hình:** MVC (Model-View-Controller)
- **Rendering:** Server-Side Rendering
- **Quản lý dự án:** Maven
- **Cơ sở dữ liệu:** MySQL
- **JDK 17**
